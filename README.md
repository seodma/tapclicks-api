# Tapclicks::Api
A simple ruby wrapper for the Tapclicks API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tapclicks-api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tapclicks-api
